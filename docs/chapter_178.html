Furthermore, there was a significant difference between heroes and regular soldiers. Beside the skill bar on the right, there was also an Item inventory. Although there were no items yet, it was clear that the game supported acquiring and using items. However, it seemed only heroes could use them.

In the past, games like Human Wars also had some special units, such as siege beasts, which were strategic control units capable of turning the tide of battle.
